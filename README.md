# Projeto FullStack SaaS

### Comandos utilizados

- Criando projeto com next js:
  `npx create-next-app@latest odontopro`

- Would you like to use the recommended Next.js defaults?
- No, customize settings (Escolha essa opção)

- Essa para criar já com typescript: Yes

- Would you like to add TypeScript with this project? Yes​

- Essa para iniciar com Eslint: No​

- Would you like to use ESLint with this project? No

- Would you like to use React Compiler? No

- Essa caso queira usar Tailwind: Yes

- Would you like to use Tailwind CSS with this project? Yes

- Would you like your code inside a `src/` directory? Yes

- Use App Router (recommended)? >> Yes

- Pergunta para usar TurboRepo pode colocar NO
- Would you like to customize the import alias (`@/*` by default)? Yes
- What import alias would you like configured? @/\*
- Would you like to include AGENTS.md to guide coding agents to write
  up-to-date Next.js code? No

- Entrando na pasta do projeto:
  `cd odontopro`
- Rodando o projeto:
  `npm run dev`

- Instalando Shadcn:
  `npx shadcn@latest init`
